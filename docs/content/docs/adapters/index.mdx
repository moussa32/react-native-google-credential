---
title: Adapters
description: Integrate Google credentials with third-party auth and backend providers.
---

Adapters help connect Google ID tokens from this package to third-party
authentication and backend providers.

The core package only acquires the Google credential. Each adapter documents how
to exchange or forward that credential to a provider-specific API.
